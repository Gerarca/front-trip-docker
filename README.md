# trip-app-front
This project provides a visual travel route


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### test
```
npm run test:unit 
```

### docker
```
docker compose build 
docker compose up -d
```
or 
```
docker compose up -d --build 
```
